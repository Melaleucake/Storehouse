## linux 重启

`shutdown -r`

## git 获取[nick name]的 1 天的 log 记录

`git log --pretty=format:"%ai , %an: %s" --since="1 day ago" | grep "[nick name]" >> ~/Desktop/file-name.log`

## 下载视频 m3u8

`ffmpeg -i [url] -c copy -bsf:a aac_adtstoasc [output_name].mp4`
